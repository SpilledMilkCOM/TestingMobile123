Loading a Picture
-----------------

> Uses the FileOpenPicker(), StorageFile.OpenAsync(), and BitmapImage.SetSourceAsync() to load a picture
> and assign it to a property to bind to the UI.


Debug on Windows 10 Phone - Set configuration from x86 to ARM.
	Then "Play/Run" on the device.
	Device must be unlocked
	Turn ON Developer mode under Settings "For Developers"
